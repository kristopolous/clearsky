Clearsky is a desktop application that makes self-hosting accessible to non-technical users. Built with Electron and Nix, it provides a guided migration wizard that walks users through moving their data from cloud services like Google Photos and Google Drive to self-hosted alternatives. The application bundles all necessary dependencies—including Podman for container orchestration, immich-go for photo imports, and tailscale for remote access—into a single AppImage that runs on any Linux system without requiring system-wide installations. Users simply download the AppImage, double-click to launch, and follow the intuitive interface to export their data from cloud providers and import it into local services, all while maintaining full control over their files and privacy.

For the NixOS Hackathon, Clearsky demonstrates Nix's power in creating reproducible, self-contained desktop applications. The Nix flake defines all dependencies declaratively, ensuring identical builds across different machines and eliminating the "works on my machine" problem. When built, the AppImage contains everything needed to run the application, from the Electron runtime to the container tools, making distribution trivial—just share one executable file. This approach aligns with the hackathon's theme of closing the gap between Nix's capabilities and everyday usability, showing how complex technical infrastructure can be hidden behind a simple user interface that anyone can use to reclaim their data sovereignty.
