Clearsky solves the problem of cloud dependency by making it effortless for non-technical users to reclaim their data from services like Google Photos, Google Drive, and iCloud. The core issue isn't just privacy—it's that ordinary users feel trapped: they know cloud services harvest their data, but self-hosting seems too complex. Clearsky removes this barrier with a modular migration framework built on Nix flakes, where each migration (like "Google Photos → Immich") is a self-contained, reusable component. Users download a single AppImage, select their migration, and follow a guided wizard that handles everything from exporting data to importing it into self-hosted alternatives—all without touching a terminal or editing config files.

This approach transforms an intimidating technical process into something as simple as installing any other desktop application. By bundling the migration framework, container orchestration (Podman), and import tools (immich-go) into one reproducible package, Clearsky eliminates the fragmentation that typically blocks users: no more hunting for tutorials, debugging Docker commands, or worrying about system compatibility. The result is a path to data sovereignty that feels normal rather than exceptional—users migrate their photos, documents, and files to self-hosted services in minutes, gaining privacy and control without sacrificing usability. For the NixOS Hackathon, this demonstrates how Nix's reproducibility and modularity can solve real-world problems: making self-hosting accessible to families, small businesses, and anyone who wants their data under their own control.
