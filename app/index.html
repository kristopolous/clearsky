✅ Recommended: Automatic Download via API

Get an API key from Google and Clearsky will automatically download all your photos. No manual export needed!

Get Your API Key:

  1. Go to Google Cloud Console
  2. Create a new project (or select existing)
  3. Enable Google Photos Library API
  4. Go to APIs & Services → Credentials
  5. Click + CREATE CREDENTIALS → API Key
  6. Copy the key (starts with AIzaSy) and paste below

📖 Full setup guide with screenshots

🔒 Key is used locally only, never sent to Clearsky servers

Having trouble with the API?

📦 Google Takeout (Manual Export)

If the API isn't working for you, you can manually export from Google Takeout:

  1. Go to takeout.google.com
  2. Click "Deselect all", then select only "Google Photos"
  3. Click "Next step" → Choose "Send download link via email"
  4. Download the ZIP file when ready (may take hours for large libraries)
  5. Return here and upload the ZIP file

📄 Google Docs Export

For Google Docs, use Google Takeout:

  1. Go to takeout.google.com
  2. Deselect all, then select only "Drive" or "Docs"
  3. Choose format: HTML or plain text
  4. Download your ZIP file and return here
